Madrid Flower Bloom Radial

A small visual project exploring seasonal flower bloom patterns in Madrid. This repository contains multiple experiment versions so I can compare design, structure, and interaction choices over time.

exp_0

Original all-in-one prototype

The original version, where layout, styling, data, and JavaScript logic are all contained inside a single HTML file.

exp_1

Data-separated version

A more flexible structure where the flower data lives in a separate JSON file and the graph logic is moved into JavaScript.
